# Primeiros Comandos com o Git :computer:

- git init
- git add
- git commit -m "*mensagem do commit*"
- git status
- echo > *nome e extensao do arquivo*
- git config --list
- git config --global --unset     user.name
- git config --global --unset     user.email

 

- git config --global     --unser.name "NOME"

 

- git remote add *origin* url em questao (origin é um apenas um *alias*, ele pode ter qualquer nome)
- git remote -v (Mostra para  quais repositórios remotos esse repositório está apontado)
- git clone
- git push *origin* master ou main
- git pull *origin* master ou main
